# Anime Buttons V1.1.0

Projeto desenvolvido para praticar:

- HTML
- CSS
- JavaScript
- Git
- GitHub

## Objetivo

Criar um site interativo baseado em animes, onde cada botão altera o tema e as informações da página.

## Funcionalidades

- Tema dinâmico
- Sistema modular
- Múltiplos animes
- Troca de informações em tempo real

## Animes atuais

- Naruto
- Dragon Ball
