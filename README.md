# Anime Buttons V1.0.0

Projeto simples desenvolvido para praticar:

- HTML
- CSS
- JavaScript
- Git
- GitHub

## Objetivo

Criar um site interativo baseado em animes, onde cada botão altera o visual da página e apresenta informações sobre personagens ou animes.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Estrutura do Projeto

Projeto/
│
├── index.html
├── css/
│   └── style.css
├── moduler/
│   └── naruto.js

## Funcionalidades da V1.0.0

- Alteração de tema da página
- Mudança dinâmica de texto
- Interação com botão
